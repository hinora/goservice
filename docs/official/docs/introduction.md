# Introduction
Goservice is a microserivce framework with golang. Goservice will help you build a system scalable, high performance.
## Features
- Event driven architecture
- Pluggable loggers
- Pluggable transporters 
- Built-in tracing feature with exporters
- Service registry & dynamic service discovery
- Multiple serivces on a node
- Master-less architecture, all nodes are equal
- Service, action, event concept
